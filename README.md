# Chat-U 💬

A real-time chat application that allows users to communicate freely without subscriptions.

## Features
- Real-time messaging
- No subscription required
- Secure backend
- Modern UI

## Tech Stack
- React (Vite)
- Node.js
- Express
- Socket.io
- MongoDB

## Setup

### Install dependencies
cd server && npm install
cd ../ui && npm install

### Run backend
cd server
npm start

### Run frontend
cd ui
npm run dev
